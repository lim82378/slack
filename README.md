1. 프로젝트 제목 : 협업툴 Slack 구현

2. 프로젝트 기간 : 2025년 11월 ~ 2026년 1월

3. 프로젝트 개요 : 협업툴 Slack 실제 구현(+AI 기능)

4. 프로젝트 선정 이유 : 공부를 위한 기능들이 많이 들어있는 프로젝트가 협업툴이라고 생각하여, 조원이 써봤던 Slack을 구현하기로 하였습니다.

5. 프로젝트 특장점 :  
        - Slack 100% 구현
        - AI 채팅 요약
        - AI 프로필 이미지 생성
        - 실시간 채팅(WebSocket)
        - 파일 업로드(프로필 이미지) ajax 미리보기

6. 맡은 역할 : 
        - 로그인 : 로그인 체크
        - 회원가입 : 사용자 추가
        - 워크스페이스 홈 : 사용자 초대 및 검색
        - 디렉터리 : 프로필 편집
        - 채널 : 채널 생성 및 즐겨찾기 추가/제거, 채팅

7. 성장점 : ajax(비동기통신)에 대한 이해와 경험을 쌓을 수 있었고,  웹소켓을 활용한 채팅창 구현, 복잡한 관계성을 쿼리로 해결할지, 여러 개의 기능들을 서비스로 합쳐 해결할지 대한 고민을 통해 생각해 볼 수 있는 경험이 되었습니다.

8. 느낀점 : 프로젝트 하면서 시간 대비 화면에서의 비치는 아웃풋이 없어서 조금 힘들었는데 오류를 해결하고, 기능을 만들면서 많은 성취감을 느꼈습니다. 기능들을 더 수정하지 못해 조금 많이 아쉬움이 남고 다음 프로젝트에서는 이런 부분들을 더 신경 쓰면서 더 잘 만들고 싶은 욕망이 생겼습니다. 저와 적성에 맞다는 것을 느꼈습니다. 어려운 오류를 해결하면서 느끼는 도파민이 너무 좋았습니다.
